# autonomous-store-simulator
autonomous-store-simulator with amazon

## Get started
 > git clone \<URL\> <br/>
 > npm install<br/>
 npm run build

### 프로젝트 디렉토리
#### 웹앱 관련
* public : React Development Build With Hot-Loader
* build : React 빌드 파일
* config : React Config 파일
* scripts : React 스크립트
* src : App개발 환경
* .env : 환경변수 설정 파일 (dotenv)

#### 서버 관련
* server
 * lambda : 람다함수들
 * 또 뭐필요하지

### 공통 
* s3 : Amazon S3의 Root Directory
